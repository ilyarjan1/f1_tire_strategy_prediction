# F1 Strategy ML Report Template

## 1) Problem
What decision are you predicting, and why does it matter?

## 2) Data
- Data source
- Rows / columns
- Target column
- Numeric vs categorical features
- Class balance

## 3) Exploration
- Key plots
- Main patterns
- Data quality issues

## 4) Preprocessing
- Missing value handling
- Encoding
- Scaling (if needed)
- Train/test split strategy

## 5) Models tested
List all models tried and why:
- RandomForest
- HistGradientBoosting
- (Optional) other models

## 6) Evaluation
- Accuracy
- Macro F1
- Precision/Recall
- Confusion matrix
- Error analysis

## 7) Best model
- Which model won?
- Why did it perform best?
- What trade-offs remain?

## 8) Improvements
- Better features
- Hyperparameter tuning
- Time-aware split
- Calibration or threshold tuning

## 9) Final conclusion
Short practical summary and next steps.
